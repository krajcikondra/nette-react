Nette-React example project
=================

* Clone repository

* Create databae named 'nette-react'

* In app/config folder create *config.local.neon* from *config.local.neon.example*

* In terminal run:

  * ```composer install```

  * ```npm install```

  * ```vendor/bin/phinx migrate```


# Launh application

In terminal run  ```npm start```
